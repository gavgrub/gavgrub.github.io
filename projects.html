Impartial Predjudice

A project built for the Hack the Bias 2026 hackathon. The hackathon was built around the idea of developing software to combat bias in a given area of society. My group (not taking the prompt seriously) decided that instead of reduce bias, we should develop software to maximize bias, arguing that it would display where bias exists online. As a result, Impartial Predjudice was born.

Using automated interaction on TikTok, it prompts the user for a target concept (dogs, music, politics) and proceeds to scroll on TikTok, analyzing video metadata and visuals with AI, scoring each video's relevance, and selectively engaging with aligned content. Over time, this skews the feed towards or away from a single viewpoint, showing how opposing "noise" is gradually silenced. The project highlights the mechanisms behind algorithmic bias and has implications for education, research, and policy.
